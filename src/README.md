# Source Files Folder

## Required Files

Place the following extracted files from Star Citizen's `Data.p4k` here:

- **global.ini** (Required) - Main localization strings
- **vehicles.ini** (Optional) - Vehicle-specific strings

## How to Extract

1. Download a P4K extraction tool (search for "Star Citizen P4K extractor")
2. Navigate to your Star Citizen installation: `StarCitizen\LIVE\Data.p4k`
3. Extract the localization files from: `Data\Localization\english\`
4. Copy `global.ini` to this folder
5. (Optional) Copy `vehicles.ini` if you want to customize vehicle text

## Note

These files are **not included** in the repository as they are copyrighted by Cloud Imperium Games. You must extract them yourself from your legal copy of Star Citizen.

Current expected version: **sc-alpha-4.3.2.10452200**

After game updates, you may need to re-extract these files to get the latest strings.
